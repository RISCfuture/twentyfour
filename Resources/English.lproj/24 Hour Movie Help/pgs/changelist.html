List of changes

Each new version of The Twenty-Four Hour Movie introduces changes: new features or bug fixes that differentiate it from the previous version. For reference, a comprehensive list of changes is included in this help file. As each new version is released, its changes are added to this file. Changes are listed in reverse chronological order, with the current (most recent) version of the program appearing first.

Version 1.0.1 (October 19, 2009)
Feature Improvements

  • If you move your image folder, or if it is on a drive that is unmounted, the program will automatically detect this and will still be able to use your image folder if possible.

Bug Fixes

  • Fixed some minor memory leaks.
