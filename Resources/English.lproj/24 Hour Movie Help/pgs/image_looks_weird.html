The desktop image looks strange

If your desktop image looks warped or distorted, you probably need to change the image settings in the application. Open The Twenty-Four Hour Movie and check that the “Scaling:” option is set to Proportional or Keep original size. (For more information, see Information on the options in the main window.)

Note that if you change this setting, it will not be reflected immediately. You will see the new setting the next time your desktop image changes.

If your image still looks weird, open the image folder in Finder and view some of the images in Preview. Be sure that they aren’t warped in Preview too. If so, your images themselves are distorted.
