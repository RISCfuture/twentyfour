The helper tool couldn’t be installed

In order to set your computer’s desktop, The Twenty-Four Hour Movie installs a helper tool in your Home > Library > Application Support > The Twenty-Four Hour Movie folder. This tool is launched periodically to set your desktop background. The Twenty-Four Hour Movie normally has permission to install files into its own helper folder.

If you suspect that the folder’s permissions were changed accidentally, one thing you can try is repairing your disk permissions.

To repair disk permissions:

  1. Open Disk Utility in Applications > Utilities, or by clicking below.

    Open Disk Utility

  2. Read Repairing disk permissions in the Disk Utility help book.
