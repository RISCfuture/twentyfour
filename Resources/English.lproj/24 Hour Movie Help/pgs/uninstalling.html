How to uninstall the program

The Twenty-Four Hour Movie can be easily removed from your computer by following these steps:

Uninstalling the program:

  1. Disable desktop changing using The Twenty-Four Hour Movie application.

    For step-by-step instructions, continue with How to turn off the program and use System Preferences to control your desktop like normal.

  2. In the Finder, open the folder containing the application. Drag the application to the Trash.

    The application is typically located in the Applications folder in your hard disk. If you do not know where the program is located, and the program is currently running, command-click its icon in the Dock and choose Options > Show In Finder.

This is sufficient to remove the functionality of the program from your computer; however, if you would like to remove all files installed by the program, continue below:

Removing all program files:

  1. In the Home > Library > Application Support folder, remove the Twenty-Four Hour Movie folder.

  2. In the Home > Library > LaunchAgents folder, remove the org.tmorgan.TwentyFourHourMovie.plist file.

  3. In the Home > Library > Preferences folder, remove the org.tmorgan.TwentyFourHourMovie.plist file.
