Couldn’t write the launch agent property list file

In order to set your computer’s desktop, The Twenty-Four Hour Movie creates a file in your Home > Library > LaunchAgents folder. This file is called “org.tmorgan.TwentyFourHourMovie.plist”. The Twenty-Four Hour Movie normally has permission to create and edit this file.

If you suspect that the folder’s permissions were changed accidentally, one thing you can try is repairing your disk permissions.

To repair disk permissions:

  1. Open Disk Utility in Applications > Utilities, or by clicking below.

    Open Disk Utility

  2. Read Repairing disk permissions in the Disk Utility help book.
